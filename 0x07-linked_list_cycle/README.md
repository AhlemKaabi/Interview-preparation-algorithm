# Linked list cycle

### **Problem statement**

Write a function in C that checks if a singly linked list has a cycle in it.

* Prototype: `int check_cycle(listint_t *list);`
* Return: 0 if there is no cycle, 1 if there is a cycle.


### **Compilation & Output**


